# stcp-explorer

STCP bus network explorer with routes, stops and live positions via GTFS static data and stcp-proxy.

**URL:** https://nunoptx.github.io/stcp-gtfs-analyzer

## Content

- `index.html` — main app
- `shapes.txt` — route shapes (GTFS)
- `stops.txt` — stop data (GTFS)

## What it shows

- Routes and variants
- Stops and realtime
- Real-time bus positions via [stcp-proxy](https://nunoptx.github.io/stcp-explorer/)
